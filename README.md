# Smart File Organizer

A Python GUI application that automatically organizes files in the Downloads folder by file type and date.

## 🔥 Features
- Organize files by type (Images, Videos, Documents, Music)
- Sort files by date (Year-Month)
- Simple GUI (Organize + Unorganize)
- One-click automation

## 🛠️ Tech Stack
- Python
- Tkinter
- OS, Shutil

## ▶️ How to Run
1. Install Python
2. Download this project
3. Open terminal in project folder
4. Run:
   python main.py

## 📂 Example
Before:
Downloads → messy files

After:
Downloads/
  Images/2026-04/
  Videos/2026-04/
  Documents/2026-04/

## 💡 Use Case
Useful for automating file management in offices, students, and content creators.
